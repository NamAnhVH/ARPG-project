# ARPG-project
## Hướng dẫn cài đặt
- Tải mã nguồn về máy
- Tải phần mềm Godot Engine https://godotengine.org/
- Mở Godot, import folder mã nguồn

## Hướng dẫn export sản phẩm
- Nhấn Project => Export
- Nếu chưa có Option export, nhấn Add... => Windows Desktop
- Chọn Option, chọn đường dẫn để xuất file
- Nhấn Export Project...
